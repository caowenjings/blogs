# git 常见问题梳理

# 1、本地项目上传远程仓库

1.远程创建项目
2.复制项目仓库地址
3.本地项目右键打开 git bash 允许命令
  git init //初始化本地仓库，运行完成你得文件夹下会多出一个.git 文件夹
4.将本地需要提交的代码粘贴到此文件夹中 运行第二个命令
  git  status
  git add --all
5.提交
  git commit -m ''
6.推送
  git push --set-upstream 远程仓库地址 分支名字
  eg：
  git push --set-upstream http：//111.111.111 master
  完成了会输入 git 账号和密码

# 2、把一个分支的修改更新到另外一个分支

1.可以使用git log命令或去gitlab界面找commit的ID号，比如026d0203
2.切换到目标分支上(即在要合并到的分支上)，执行命令：
  git cherry-pick aabb4d0203

# 3、查看分支

1.查看当前分支
  git branch
2.查看所有分支详情
  git branch -av
